## This is a simple CRUD project.

A CRUD(Create, Read, Update and Delete) simple project. Made for studies purposes

For this project I used: 

✔️JavaScript
✔️PostgreSQL
✔️Node
✔️Express
✔️Nunjucks

## ⚙ Configuration

1- If you wanna to install dependencies:
> npm install

2- To start the application:
> npm start
